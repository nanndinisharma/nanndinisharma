Smash Badminton Club

Website Pitch Deck

Our Idea

A modern badminton club website showcasing schedules, team info, and events, helping players connect, join easily, and stay updated online..

Features

The website includes an interactive carousel, easy navigation menu, membership form, location details, and social media links for user engagement.

Technology Used

This website was developed using HTML for structure, CSS for styling, and JavaScript for interactivity. Bootstrap was used to create a responsive and modern design.

Future Plans

In the future, the website can include online booking, tournament updates, player profiles, and expanded features for better user interaction and engagement.
